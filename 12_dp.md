1. *1137. N-th Tribonacci Number*
The Tribonacci sequence Tn is defined as follows: 

$T_0 = 0, ~ T_1 = 1, ~ T_2 = 1, ~ and ~ T_{n+3} = T_n + T_{n+1} + T_{n+2} ~ for ~ n >= 0$

Given n, return the value of `Tn`.

> O(LogN): State Transition + Matrix exponential
```python
class Solution:
    def tribonacci(self, n: int) -> int:
        if n == 0:
            return 0
        if n == 1:
            return 1
        if n == 1:
            return 1
        M = [[1, 1, 1], [1, 0, 0], [0, 1, 0]]
        I = self.power_n(M, n - 2)
        res = 0
        res += (I[0][0] * 1 + I[0][1] * 1)
        return res

    def multiply(self, A, B):
        res = [[0] * 3 for _ in range(3)]
        for i in range(3):
            for j in range(3):
                for k in range(3):
                    res[i][j] += A[i][k] * B[k][j]
        return res

    def power_n(self, A, n):
        I = [[0] * 3 for _ in range(3)]
        for i in range(3):
            I[i][i] = 1
        
        while n > 0:
            if n % 2 != 0:
                I = self.multiply(I, A)
            A = self.multiply(A, A)
            n >>= 1
        return I
```
2. *House Robber*
- You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security systems connected and it will automatically contact the police if two adjacent houses were broken into on the same night.

- Given an integer array nums representing the amount of money of each house, return the maximum amount of money you can rob tonight without alerting the police.

```python
def rob(nums: List[int]) -> int:
        n = len(nums)
        if n == 1: return nums[0]
        dp = [0] * n 
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, len(nums)):
            dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
        return dp[n - 1]
```

3. *Unique Paths II*
A grid may have many obstacles, indicated by `1` in `obstacleGrid`, 
Example: 
`Input: obstacleGrid = [[0,0,0],[0,1,0],[0,0,0]]
Output: 2`
Explanation: There is one obstacle in the middle of the 3x3 grid above.
There are two ways to reach the bottom-right corner:
- Right -> Right -> Down -> Down
- Down -> Down -> Right -> Right

> Initialization: from bottom right corner, if 1 appears, then any other dp on the left of it should be 0
```python
def uniquePathsWithObstacles(obstacleGrid: List[List[int]]) -> int:

        m = len(obstacleGrid)
        n = len(obstacleGrid[0])

        if obstacleGrid[m-1][n-1] == 1:
            return 0

        dp = [[0] * n for _ in range(m)]
        dp[m-1][n-1] = 1

        for i in range(n - 2, -1, -1):
            if obstacleGrid[-1][i] == 1:
              break
            else:
                dp[-1][i] = 1

        for j in range(m - 2, -1, -1):
            if obstacleGrid[j][-1] == 1:
                break
            else:
                dp[j][-1] = 1

        for i in range(m - 2, -1, -1):
            for j in range(n - 2, -1, -1):
                if obstacleGrid[i][j] == 1:
                    dp[i][j] = 0
                else:
                    dp[i][j] = dp[i + 1][j] + dp[i][j + 1]
        return dp[0][0]
```

```python
def longestPalindrome(s: str) -> str:
        n = len(s)
        if n < 2:
            return s

        dp = [[False] * n for _ in range(n)]
        start = 0
        max_len = 1

        for r in range(n):
            for l in range(r + 1):
                if s[l] == s[r] and (r - l <= 2 or dp[l + 1][r - 1]):
                    dp[l][r] = True
                    if r - l + 1 > max_len:
                        max_len = r - l + 1
                        start = l
        return s[start:start + max_len]
```

5. *377. Combination Sum IV*
- Example 1:
`Input: nums = [1,2,3], target = 4`
`Output: 7`
Explanation:
The possible combination ways are:
(1, 1, 1, 1)
(1, 1, 2)
(1, 2, 1)
(1, 3)
(2, 1, 1)
(2, 2)
(3, 1)
Note that different sequences are counted as different combinations.

```python
def combinationSum4(nums: List[int], target: int) -> int:
        from functools import lru_cache
        @cache
        def dfs(t):
            if t == 0:
                return 1
            if t < 0:
                return 0
            ways = 0
            for n in nums:
                ways += dfs(t - n)
            return ways 
        return dfs(target)
```
6. *91. Decode Ways*

- Example 1:

`Input: s = "226"`
`Output: 3`
Explanation:
"226" could be decoded as "BZ" (2 26), "VF" (22 6), or "BBF" (2 2 6).

- Example 2:

`Input: s = "06"`

`Output: 0`

Explanation:
"06" cannot be mapped to "F" because of the leading zero ("6" is different from "06"). In this case, the string is not a valid encoding, so return 0.

```python
def numDecodings(s: str) -> int:
        if s[0] == '0':
            return 0
        n = len(s)
        dp = [1] + [0] * (n - 1)
        for i in range(1, n):
            if s[i] != '0':
                dp[i] = dp[i - 1]
            if s[i - 1] != '0' and 10 <= int(s[i - 1 : i + 1]) <= 26:
                if i == 1:
                    dp[i] += 1
                else:
                    dp[i] += dp[i - 2]
        return dp[-1]
```
7. *343 Integer Break*

Given an integer n, break it into the sum of k positive integers, where k >= 2, and maximize the product of those integers.
Return the maximum product you can get.
- Example:
Input: n = 10
Output: 36
Explanation: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36.

```python
def integerBreak(n: int) -> int:
        dp = [0] * (n + 1)
        dp[1] = 1
        for i in range(2, n + 1):
            for left in range(1, i):
                dp[i] = max(dp[i], left * (i - left), left * dp[i - left])   
        return dp[n]
```
